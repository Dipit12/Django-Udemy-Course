The Document Object Model(DOM) allows us to interface out Javascript code
to interact with HTML and CSS

Browser will construct the DOM, which basically means storing all the HTML tags as javascript objects

Some important document attributes:
1. document.URL
2. document.body
3. document.head
4. document.links

Methods for grabbing elements from the DOM
1. document.getElementById()
2. document.getElementByClassName()
3. document.getElementByTagName()
4. document.querySelector()
5. document.querySelectorAll()
